# AI 你画我猜网页游戏 - 项目任务清单

## 项目概述
创建一个在线你画我猜网页游戏，玩家可以在游戏中绘制图片，AI 系统会根据玩家的绘制内容进行猜测并输出猜测结果。

## 技术栈
- 脚手架：Vite
- 前端框架：React + TypeScript
- 画布：HTML5 Canvas
- 后端：Node.js + Express
- AI 集成：OpenAI Vision API
